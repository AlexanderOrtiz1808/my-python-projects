# 🚀 My Python Projects

Hello! Welcome to my repository. Here, I am documenting my learning journey and evolution as a future Software Engineer, building projects completely from scratch.

## 📂 Developed Projects

### 1. Mini Trivia Game (V1 & V2)
An interactive general knowledge Q&A game played directly from the terminal.
* **Applied Concepts:** Variable handling, `inputs`, conditionals (`if/elif/else`), Data Structures (Lists and Tuples), `for` loops, and user text sanitization using methods like `.lower()` and `.strip()`.

### 2. Basic Expense Calculator
A terminal-based financial tool to track purchases that validates the available budget in real-time to prevent negative balances.
* **Applied Concepts:** Infinite loops (`while`), data type casting (`int()`), assignment operators (`-=`), loop interruptions (`break`), and logical data validation before mutating state.

### 3. Rock, Paper, Scissors vs PC (V1 & V2)
A classic console game featuring an infinite tournament mode and a live scoring system.
* **Applied Concepts:** The `random` library to generate computer choices, strict boolean logic (`and`), infinite loops (`while True`), control flow interruptions (`break`), and state management using score counters (`+= 1`).
