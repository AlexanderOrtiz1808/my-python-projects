## 📂 What's in this repository?

Currently, you can find my core projects and their iterations:

* 🎮 **Mini Trivia Game (General Knowledge):** An interactive terminal-based game.
    * **`Mini-trivia-v1.py`**: My original code, where I designed the points accumulators, conditionals, and logic from scratch.
    * **`Mini-trivia-v2.py`**: An optimized version of the same game, applying code refactoring, the DRY (Don't Repeat Yourself) principle, lists, and `for` loops for cleaner, more scalable code.

* 🪨📄✂️ **Rock, Paper, Scissors Game:** A classic game built using flow control and automation.
    * **`piedra_papel_tijeras_v1.py`**: Basic structure using logic comparisons and the `random` module.
    * **`piedra_papel_tijeras_v2.py`**: Enhanced version featuring a continuous gameplay loop, score tracking, and an exit trigger.

* 💰 **Budget Tracker & Expense Calculator:** Financial tools designed to handle user inputs and arithmetic operations.
    * **`calculadora-gastos-v1.py`**: Tracks daily expenses and updates the user's available balance.
    * **`calculadora-gastos-v2.py`**: Added a safety layer with validation logic to prevent overspending beyond the budget limits.

* 📊 **Mini Data Analyzer:** A script simulating data analysis techniques.
    * **`mini_data_analyzer.py`**: Generates datasets dynamically using random generation and processes the information using Python's built-in mathematical functions (`max`, `min`, `sum`, `len`) to calculate metrics and averages inside a structured menu loop.
